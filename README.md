# SEO Knowledge Base

SEO 需求挖掘 → 机会验证 → AI 建站 → 反馈闭环 知识库

## 目录结构

| 目录 | 用途 |
|------|------|
| `niches/` | 赛道评估（每个赛道一个文件，含评分、决策、理由） |
| `opportunities/` | 验证通过的关键词机会 |
| `backlinks/` | 可复用外链资源库（按行业分类） |
| `playbooks/` | 成功模式 & 失败教训 |
| `sites/` | 已部署站点追踪（域名、关键词、排名变化） |
| `signals/` | 需求信号存档（按月归档） |
| `pipelines/` | SEO pipeline 运行结果 |

## 工作流

```
RSS/社交信号 → 需求提取 → SEO pipeline 验证 → 赛道评分 → 建站 → 排名追踪 → 回哺知识库
```
